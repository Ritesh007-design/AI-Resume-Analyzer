# AI Resume Analyzer

AI-powered Resume Analyzer built using Streamlit and Groq LLM.

## Features

- Multi-Role Resume Analysis
- ATS Score Evaluation
- Resume Keyword Match Score
- Skill Gap Analysis
- AI-Powered Suggestions
- Interview Preparation Tips
- Hiring Readiness Score

## Tech Stack

- Python
- Streamlit
- Groq LLM
- PyPDF2

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
